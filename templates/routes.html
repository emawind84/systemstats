List All The Routes Defined

This is a list of the defined routes on the application. They've been passed to the template after retrieving the list using this code:
    {% for route in routes %}
  • {{route}}
    • {% endfor %}

Code to retrieve the routes

for rule in app.url_map.iter_rules():
# Filter out rules we can't navigate to in a browser
# and rules that require parameters
if "GET" in rule.methods and len(rule.defaults) >= len(rule.arguments):
    url = url_for(rule.endpoint)
    links.append((url, rule.endpoint))
# links is now a list of url, endpoint tuples
